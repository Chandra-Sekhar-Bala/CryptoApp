# CryptoApp
Track all the Crypto Currency in a single app. Organise by CMC_Rank, click filter and choose your favorite sort method.

Features
- Splashscreen
- Crypto Tracker
- Sort by 
   - CMC
   - Price
   - up-down in 24 hr
